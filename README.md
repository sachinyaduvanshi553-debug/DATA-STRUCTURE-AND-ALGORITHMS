## 🚀 Quick Start

</div>

```bash
# ✅ Ensure JDK 17+ is installed
git clone https://github.com/sachinyaduvanshi553-debug/DATA-STRUCTURE-AND-ALGORITHMS.git
cd DATA-STRUCTURE-AND-ALGORITHMS-main

# Compile all source files
javac src/**/*.java

# Run the demo driver
java -cp src Main
```

> **Maven users:**
> ```bash
> mvn compile
> mvn exec:java -Dexec.mainClass=Main
> ```
